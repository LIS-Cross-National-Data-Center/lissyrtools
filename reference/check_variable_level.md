# Checks if the variable is 'household' or 'person' level.

**\[deprecated\]**

## Usage

``` r
check_variable_level(variable)

safely_check_variable_level(...)
```

## Arguments

- variable:

  A String with the name of the variable. E.g. 'pi11', 'dhi'.

## Details

Throws an error if it couldn't be checked.
