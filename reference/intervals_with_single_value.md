# Interval has single value

Check if interval beginning and end are equal and thus it's composed of
a single value.

## Usage

``` r
intervals_with_single_value(x)
```

## Arguments

- x:

  string with the interval.

## Value

A boolean indicating if the first and second element of the interval are
the same value.

## Details

To be used inside 'compute_lorenz_curve()'
