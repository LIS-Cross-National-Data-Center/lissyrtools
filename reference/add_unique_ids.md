# Add unique ID variables when binding LIS or LWS files.

Add unique ID variables when binding LIS or LWS files.

## Usage

``` r
add_unique_ids(file, file_name, file_level)
```

## Arguments

- file:

  A LIS or LWS file.

- file_name:

  The name of the LIS or LWS file.

- file_level:

  A string indicating the level of the file. Valid inputs are:
  'household', 'h', 'person' or 'p'.
