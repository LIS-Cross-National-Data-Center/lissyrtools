# Set lissy attributes

Sets the 'level', 'merged_levels' and 'database' attributes of an object
equal to the specified ones.

## Usage

``` r
set_lissy_attributes(lissy_object, lissy_attributes)
```

## Arguments

- lissy_object:

  A list of tibbles or a tibble.

- lissy_attributes:

  A list with three elements: 'level', 'merged_levels' and 'database'.

## Value

The same type of object as 'lissy_object'.
