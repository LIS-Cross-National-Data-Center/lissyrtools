# lissyrtools: Functions that can be used within LISSY

A package with the tools needed to develop scripts with LIS data. It
allows users to:

- Read LIS data within the LISSY environment, or to use LIS sample files
  locally.

- Carry out commonly performed data cleaning tasks.

- Compute estimates from microdata.
