# Rearranges a list whose elements are ccyy's into country names with numeric vectors names after its years.

Rearranges a list whose elements are ccyy's into country names with
numeric vectors names after its years.

## Usage

``` r
convert_list_from_ccyy_to_cc_names_yyyy(data_list)
```

## Arguments

- data_list:

  A list

## Value

A list
