# Checks the share of missing values for each column acroos a list of data frames.

Checks the share of missing values for each column acroos a list of data
frames.

## Usage

``` r
check_missing_share(data_list, include_zeros = FALSE)
```

## Arguments

- data_list:

  A list

- include_zeros:

  Logical. If `TRUE`, values equal to zero are also accounted in the
  percentage of missings.

## Value

A list
